# HSE ML PRo 1

Был проделан полный цикл работы с данными для задачи предсказания стоимости автомобилей, включающий:

 - Исследование данных (EDA): проверка пропусков, дубликатов, анализ статистик числовых и категориальных признаков, визуализация распределений и корреляций.

 - Очистку и подготовку данных: преобразование строковых признаков в числовые (удаление единиц измерения, парсинг сложных признаков типа torque), заполнение пропусков медианами, удаление дубликатов.

 - Выделение признаков и кодирование категориальных переменных с помощью One-Hot Encoding, при этом учитывали необходимость удаления одного столбца для устранения мультиколлинеарности.

 - Обучение модели регрессии (линейной и Ridge) с подбором гиперпараметров (альфа) через кросс-валидацию и GridSearchCV для улучшения качества.

 - Создание и применение кастомных бизнес-метрик, учитывающих допустимые погрешности и штрафующих недопрогнозы сильнее перепрогнозов, что важно для реального бизнеса.

 - Итоговый анализ моделей с точки зрения метрик качества и бизнес-ценности.
